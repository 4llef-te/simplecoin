# SimpleCoin

SimpleCoin is a light, item-based currency mod for SMPs and modpacks, with banking features.

---

## Features

- 8 coin types:
  - Gold Coin
  - Copper Coin
  - Diamond Coin
  - Emerald Coin
  - Iron Coin
  - Netherite Coin
  - Continental Coin
  - Galactic Credit
- Persistent banking system â€” balances are saved to disk and survive server restarts.
- Commands:
  - `/deposit <coin|all> <amount|all>`
  - `/withdraw <coin|all> <amount|all>`
  - `/balance`
  - `/transfer <target> <coin|all> <amount|all>`
